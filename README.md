## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

## Prerequisites

- **Ruby version**: `3.1.2`
- **Rails version**: `7.2.2`
- **Database**: PostgreSQL

## Installation

1. Clone the repo:
```bash
 https://github.com/sid9568/file_uploader.git
```
2. Step
```bash
 rails db:create
```
3. Step
```bash
 rails db:migrate
```
5. Step
```bash
 rails assets:precompile
```
6. Step
```bash
 rails s
```
